# A music player (wip)

This project targets the [ULX3S board](https://radiona.org/ulx3s/) with a [SSD1351 128x128 pixel OLED screen](https://www.waveshare.com/1.5inch-rgb-oled-module.htm).

## Testing

- `make ulx3s`

## Exercises

- Add a memory mapping to buttons.
